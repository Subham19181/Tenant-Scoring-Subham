# 🏠 Tenant Scoring DApp (Soroban)

## 📌 Project Description

Tenant Scoring is a fully permissionless decentralized application built on Stellar Soroban. It enables landlords, roommates, and property managers to share and access reputation scores of tenants in a trustless and transparent way.

Unlike traditional systems, this protocol does not rely on centralized authorities — anyone can submit and view tenant reviews.

## ⚙️ What It Does

- Allows any user to submit a score (1–10) for any tenant address
- Stores all reviews on-chain using Soroban smart contracts
- Computes the average score of a tenant dynamically
- Maintains a transparent and tamper-proof reputation system

---

## 🚀 Features

- 🔓 **Permissionless by design**
  - No admin roles
  - No restricted functions
  - Anyone can review any tenant

- 📊 **On-chain reputation system**
  - All reviews stored on Stellar blockchain
  - Fully transparent and verifiable

- 🧮 **Dynamic scoring**
  - Real-time average score calculation

- 🧑‍🤝‍🧑 **Decentralized identity**
  - Uses Stellar addresses as tenant identifiers

- ⚡ **Lightweight and scalable**
  - Minimal storage structure
  - Efficient retrieval of reviews
